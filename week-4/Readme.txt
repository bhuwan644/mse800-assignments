This is a yoobee college project

##Description

This system involves three main actors: Students, Lecturers, and Administrators.

Students enroll in programs, make payments, attend lectures, and submit assignments.

Lecturers deliver lectures, mark assignments, and provide feedback.

Administrators manage payments, class sessions, lecturer allocation, and student records.


##Actors

1. Student
2. Administrator
3. Lecturer
 


##Use-cases for this project

Student-> Enrolls program
Student-> Make payment
Administrator-> Manage payment
Student-> Attend lectures
Student -> Submits assignment
Lecturer-> Marks assignment
Administrator-> Manage class sessions
Lecturer-> Provides lectures
Lecturer-> Provide feedback
Administrator-> Manage lecture allocation
Administrator-> Maintain student records
