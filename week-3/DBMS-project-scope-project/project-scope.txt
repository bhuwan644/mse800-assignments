
Yoobee college offers different programs to both the domestic and international students. 
They have top notch lectures assigned in different programs. Each program contains 
around 8-10 course module and students can enroll into these course in 3 main intakes, July, October and February. Enrollment process includes students providing their entire personal details including their personal academia background. Once the program starts, there are multiple lecturers teaching different subjects in the course program and students are assigned assignments on weekly basis and the assignment results are stored in the assignment records 

So this proposed database will manage information on students, lecturers, course modules , enrollments and also the assignment records.

Possible Table and Features for this project scope\

1. Student : ID, Name, Address, contact, intake......
2. Course : ID, course name, lecturer.....
3. Lecturer : ID, name, course_teaching, contact, address.....
4. Assignment Records: Assignment_Id, Linked_course, Student_Id........
